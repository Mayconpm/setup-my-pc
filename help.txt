Usage:
	smpc [ACTION]
	smpc [ACTION] [OPTION]

Action:
	install 	Install the software related to the [OPTION] chosen
	version 	Show last version of smpc
	help 			Show this help message

Option:
	basic 		Install basic tools
	personal 	Install jonaselan personal tools
	dev 			Install development tools

Example:
	smpc install personal
